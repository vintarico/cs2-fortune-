3. SISTEMA PROVABLY FAIR
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Componentes:
├── Server Seed (hash público antes da abertura)
├── Client Seed (escolhido pelo usuário)
├── Nonce (contador de aberturas)
├── Resultado = Hash(ServerSeed + ClientSeed + Nonce)
└── Verificação pública do resultado

Interface na abertura:
┌─────────────────────────────────────────────────────────────────────┐
│  🎲 PROVABLY FAIR                                                   │
│  Server Seed (SHA256): a7b3c4d5... [Verificar]                     │
│  Client Seed: [seu-seed-aqui] [🔄 Trocar]                          │
│  Nonce: #1234                                                       │
│  ✅ Este resultado pode ser verificado                             │
└─────────────────────────────────────────────────────────────────────┘

Histórico de verificação:
- Todas as aberturas salvas com seeds
- Botão "Verificar Fairness" em cada item
- Página dedicada /provably-fair com explicação

14. PROBABILIDADES REALISTAS (Baseadas no CS:GO)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Raridade → Chance Aproximada:
├── Consumer Grade (Branco): ~79.92%
├── Industrial Grade (Azul Claro): ~15.98%
├── Mil-Spec (Azul): ~3.20%
├── Restricted (Roxo): ~0.64%
├── Classified (Rosa): ~0.32%
├── Covert (Vermelho): ~0.64%
└── Exceedingly Rare (Dourado): ~0.26% (Facas/Luvas)

Exemplo de Caixa (24 itens):
├── 12 Consumer Grade (50% cada = 79.92% total)
├── 7 Industrial Grade (2.28% cada ≈ 15.98% total)
├── 3 Mil-Spec (1.07% cada ≈ 3.20% total)
├── 1 Restricted (0.64%)
├── 1 Classified (0.32%)
└── 1 Covert ou Faca (0.64% vermelho + 0.26% faca = 0.90%)

Sistema de Roll (0-9999):
- 0-7991: Consumer Grade
- 7992-9589: Industrial Grade
- 9590-9909: Mil-Spec
- 9910-9973: Restricted
- 9974-9993: Classified
- 9994-9999: Covert (9994-9997) ou Faca (9998-9999)


15. ESTRUTURA DO BANCO DE DADOS (Prisma Schema)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

model User {
  id            String   @id @default(cuid())
  username      String   @unique
  email         String   @unique
  password      String
  avatar        String?
  balance       Float    @default(0)
  level         Int      @default(1)
  xp            Int      @default(0)
  streak        Int      @default(0)
  lastLogin     DateTime?
  createdAt     DateTime @default(now())
  
  // 2FA
  twoFactorSecret String?
  twoFactorEnabled Boolean @default(false)
  
  // Relações
  inventory     InventoryItem[]
  caseOpenings  CaseOpening[]
  battles       BattleParticipant[]
  transactions  Transaction[]
  achievements  UserAchievement[]
}

model InventoryItem {
  id          String   @id @default(cuid())
  userId      String
  user        User     @relation(fields: [userId], references: [id])
  skinName    String
  skinImage   String
  rarity      String
  value       Float
  acquiredAt  DateTime @default(now())
  sold        Boolean  @default(false)
  soldAt      DateTime?
  soldPrice   Float?
}

model CaseOpening {
  id            String   @id @default(cuid())
  userId        String
  user          User     @relation(fields: [userId], references: [id])
  caseId        String
  caseName      String
  casePrice     Float
  wonItemName   String
  wonItemValue  Float
  wonItemRarity String
  wonItemImage  String
  profit        Float
  
  // Provably Fair
  serverSeed    String
  clientSeed    String
  nonce         Int
  resultHash    String
  
  openedAt      DateTime @default(now())
}

model Battle {
  id            String   @id @default(cuid())
  mode          String   // "1v1", "2v2", "1v1v1v1"
  caseId        String
  caseName      String
  rounds        Int
  costPerPlayer Float
  isPrivate     Boolean  @default(false)
  status        String   @default("waiting") // waiting, active, finished
  winnerId      String?
  createdAt     DateTime @default(now())
  startedAt     DateTime?
  finishedAt    DateTime?
  
  participants  BattleParticipant[]
  results       BattleResult[]
}

model BattleParticipant {
  id        String   @id @default(cuid())
  battleId  String
  battle    Battle   @relation(fields: [battleId], references: [id])
  userId    String
  user      User     @relation(fields: [userId], references: [id])
  position  Int
  totalValue Float   @default(0)
  isWinner  Boolean  @default(false)
  joinedAt  DateTime @default(now())
}

model Transaction {
  id          String   @id @default(cuid())
  userId      String
  user        User     @relation(fields: [userId], references: [id])
  type        String   // deposit, withdraw
  method      String   // pix, card, steam, crypto
  amount      Float
  status      String   @default("pending") // pending, completed, failed
  metadata    Json?
  createdAt   DateTime @default(now())
  completedAt DateTime?
}

model Achievement {
  id          String   @id @default(cuid())
  code        String   @unique
  name        String
  description String
  icon        String
  rewardType  String   // balance, xp, case
  rewardValue Float
  
  users       UserAchievement[]
}

model UserAchievement {
  id            String   @id @default(cuid())
  userId        String
  user          User     @relation(fields: [userId], references: [id])
  achievementId String
  achievement   Achievement @relation(fields: [achievementId], references: [id])
  progress      Int      @default(0)
  completed     Boolean  @default(false)
  completedAt   DateTime?
  claimed       Boolean  @default(false)
  claimedAt     DateTime?
}
