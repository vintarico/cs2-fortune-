✅ 3. INFORMAÇÕES DE CADA RARIDADE
───────────────────────────────────────────────────────────────
   Cada grupo de raridade mostra:
   ✓ Nome da raridade (Legendary, Epic, Rare, etc)
   ✓ Cor exclusiva (vermelho, rosa, roxo, azul)
   ✓ Ícone visual
   ✓ % TOTAL de chance daquela raridade
   ✓ Quantidade de itens

✅ 5. INFORMAÇÕES DETALHADAS POR ITEM
───────────────────────────────────────────────────────────────
   Cada card mostra:
   ✓ Imagem da skin (Steam ou local)
   ✓ Nome da skin (truncado se muito longo)
   ✓ Arma base (Glock-18, AK-47, etc)
   ✓ % de chance individual (ex: 0.130%)
   ✓ Preço no hover
   
   Layout otimizado:
   - 2 colunas em mobile
   - 3 colunas em tablet
   - 4 colunas em desktop
