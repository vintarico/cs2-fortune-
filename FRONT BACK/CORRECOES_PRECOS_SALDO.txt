1️⃣ CARREGAMENTO DE PREÇOS:

   Quando você abre uma caixa:
   ├─ Nome da skin: "AK-47 | Redline"
   ├─ Mapeado para: "AK-47 | Redline (Field-Tested)"
   ├─ Busca na API Steam: /api/steam/price?name=...
   ├─ Retorna preço real em BRL
   └─ Exibe no modal da caixa

2️⃣ ATUALIZAÇÃO DE SALDO:

   Após abrir caixa:
   ├─ Backend desconta o valor
   ├─ Retorna novo saldo
   ├─ Frontend atualiza estado
   ├─ Salva no localStorage
   └─ Exibe novo saldo na navbar

3️⃣ FALLBACK INTELIGENTE:

   Se a API Steam falhar:
   ├─ Usa valor padrão da caixa
   ├─ Marca como "from_fallback"
   ├─ Continua funcionando normalmente
   └─ Log no console para debug
